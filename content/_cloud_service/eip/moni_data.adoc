//查看监控数据

您可以通过 EIP 的监控功能查看 EIP 实例的相关监控指标，实时监测流量波动情况，即时发现异常波动，调整带宽峰值，避免因带宽限速导致网络访问延迟。

== 前提条件

EIP 已绑定资源。

== 操作步骤

include::./common_operation.adoc/[tag=ipv4]

. 在 EIP 列表，点击 EIP 的 ID 号，进入 EIP 详情页。
. 在**监控**页签，可以查看 EIP 的流量监测数据。支持的监测项，请参见<<_监控指标,监控指标说明>>。
+
image::/images/cloud_service/network/eip/eip_monitor.png[]
+
查看方式：
+
* 点击页面右上方的时间选项，可查看**最近1小时**、**最近一天**、**最近两周**、**最近一个月**、**最近6个月**及**自定义**时间段的监控数据。
* 点击实时数据开关image:/images/cloud_service/network/eip/enable_icon.png[60,60]，可以开启实时监控数据查看。

== 监控指标

EIP 支持的监控指标如下表所示。

[cols="1,3"]
|===
|指标|说明

|流入带宽|外部访问 EIP 实例所消耗的带宽。单位：Kbps。
|流出带宽|EIP 实例访问外部所消耗的带宽。单位：Kbps。
|流入流量|外部访问 EIP 实例所消耗的流量。单位：MB。
|流出流量|EIP 实例访问外部所消耗的流量。单位：MB。
|流入包速率|EIP 实例每秒接收的数据包个数。单位：pps。
|流出包速率|EIP 实例每秒发出的数据包个数。单位：pps。
|===
