[#q1]
== EIP 流量计费的具体统计方法是什么？

如果 EIP 的计费模式是按流量计费，则除了固定的 IP 地址价格外，流量数据会每小时自动统计一次，统计方法为：对统计周期内的进、出流量求和，取其中较大的值作为扣费依据。其中，进流量费用只对超出 10Mbps 之后的流量计费。

[#q3]
== 什么情况下会收取 IP 地址附加费?

针对按带宽计费的 EIP，每个主账户名下都有免费的 IP 地址额度（默认为 2 个），当申请的 IP 地址个数超过免费额度时，系统会对额度外的每个 IP 收取固定的附加费用。该费用记在主账户名下。

[#q4]
== 按流量计费的 EIP 可以转换为按带宽计费吗？

可以。公网 IP 支持按带宽计费和按流量计费两种收费模式，两种收费模式可以互相转换。
