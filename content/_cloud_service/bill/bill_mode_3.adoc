//资源包计费

资源包（或套餐包）是一种具有时长或使用量限制的资源使用权限，属于预付费模式。用户在购买时进行付费，在资源使用过程中，优先消耗资源包中的量。

include::../../_components/note.adoc[]
本文仅对计费方式进行说明，具体价格请以产品定价为准。
include::../../_components/table_footer.adoc[]

== 适用场景

资源包通常是按量计费项的一种计费模式。适用于计费项用量相对稳定的场景。

== 计费规则

* *资源包购买*
+
用户需要以预付费的方式购买资源包，即购买时一次性支付资源包的费用。
+
----
资源包费用 = 资源包单价 * 个数 * 折扣率
----

* *资源包消耗*
+
资源包在有效期内且有剩余量值时，优先扣除资源包中的量值，资源包使用完后，按照link:../according_need/#amount[按使用量计费]的方式扣除账户余额。
