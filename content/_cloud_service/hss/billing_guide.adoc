//计费说明

本节介绍主机安全服务的计费说明。

== 计费前提

云平台配置中已启费用功能，且主机安全服务已对接费用中心，主机安全才可使用计费功能。

== 计费模式

主机安全提供**包年包月**计费模式，购买服务时，您需要预先支付购买时长内的费用。购买时长越长，优惠力度越大。

== 计费项

主机安全服务根据您购买的云服务器防护授权数、防篡改授权数、购买时长进行计费。

[cols="1,3"]
|===
|计费项目 |计费说明

|云服务器防护授权（必选）
|根据选购的授权数量计费。首次购买数量最少为 1 个，升级时可为 0 个。

|防篡改授权（可选）
|根据选购的授权数量计费。数量为 0 表示不购买，若不购买，则主机安全服务不包含防篡改功能。

|购买时长（必选）
|提供包月和包年的购买模式。

|===


== 费用计算

* 首次选购：
+
----
购买费用 = (云服务器防护授权数 × 月单价 + 防篡改授权数 × 月单价) × 购买时长 × 折扣率
----


* 升级费用：
+
----
升级费用 = (新购云服务器防护授权数 × 月单价 + 新购防篡改授权数 × 月单价) × 首次选购购买时长 × 折扣率 × 有效期比例
----

+
include::../../_components/note.adoc[]
* 升级购买时长默认与首次选购购买时长一致。
* 升级仅升级授权个数，不会延长到期时间。升级费用按照授权实际有效使用时间进行计算，有效期比例 = 到期前有效使用时间/购买时长。
include::../../_components/table_footer.adoc[]


== 续费

支持自动续费和手动续费。

* 自动续费：您可以在购买服务时开启自动续费，开启后，在余额充足的情况下，系统自动续费，续费时长与初次购买时长一致。

* 手动续费：过期前七天或过期七天以内，均可手动续费，续费时长与初次购买时长一致。过期七天以后，数据将被销毁，需要重新购买服务。续费详细操作，请参考link:../manual/renewal/[续费]。


== 到期与欠费

服务到期后，如果没有按时续费，云平台会提供 7 天的保留期，保留期内，您可随时续费使服务恢复正常，但不可进行升级、登录管理中心等操作。过期超过 7 天，资源及数据将被销毁，需要重新购买服务。

账号欠费后，无法进行续费。为防止资源不被停止或者释放，请您及时充值。
