//服务信息概览

在主机安全的**系统管理**页面，您可一览主机安全服务的基本信息。

== 操作步骤

include::./srv_entrance.adoc[]
+
**系统管理**页面包含**我的配置**及**管理中心列表**两部分。您可以在该页面查看服务相关配置信息及管理中心相关信息。

== 页面信息说明


=== 我的配置

**我的配置**展示了当前主机安全服务的防护授权总数、购买时长、创建时间、到期时间等信息。


image::/images/cloud_service/security/hss/my_cfg.png[]

各字段说明如下表所示。

[cols="1,3"]
|===
|字段 |说明

|云服务器授权总数
|已购买的云服务授权总数，为<<_管理中心列表>>中所有**云服务器防护授权**之和，包括已使用的和未使用的授权。

|防篡改授权总数
|已购买的防篡改授权总数，<<_管理中心列表>>中所有**防篡改授权**之和，包括已使用的和未使用的授权。

|购买时长
|购买服务时所选择的``购买时长``。

|续费方式
|当前服务的续费方式，可为自动续费或手动续费。 +
默认与购买服务时的设置一致，可通过link:../renewal/[自动续费]操作修改。

|创建时间
|购买/开通服务的时间。

|到期时间
|服务到期的时间。
|===


=== 管理中心列表

管理中心列表以管理中心维度，展示了每个管理中心下的防护资产、云服务器授权数、防篡改授权数以及管理中心支持的相关操作。

image::/images/cloud_service/security/hss/edr_list.png[]

列表字段说明如下表所示。

[cols="1,3"]
|===
|字段 |说明

|管理中心
|管理中心名称。 +

仅列出购买过防护授权的管理中心名称。


|资产名
a|已纳入管控的资产名称。 +

在管控资产中安装 Agent 成功后，刷新列表数据，资产名即可显示在此列。

include::../../_components/note_in_table.adoc[]
未被纳管的资产不会显示在此列，当防护授权已用完，即便在新资产中安装 Agent，新资产也不会被纳入管控。 
include::../../_components/footer_in_table.adoc[]

|云服务器防护授权
|当前管理中心已购买的云服务器防护授权数。

所有管理中心的云服务器防护授权数之和等于<<_我的配置>>中的**云服务器防护授权总数**。

授权总数不满足您的需求时，可通过link:../renewal/[升级]操作升级授权配额。

|防篡改授权
|当前管理中心已购买的防篡改授权数。

所有管理中心的防篡改授权数之和等于<<_我的配置>>中的**防篡改授权总数**。

|操作
a|管理中心操作菜单。

* 登录：点击**登录**，即可link:../login_edr/[登录管理中心控制台]。
* 代理地址：点击**代理地址**，可查看和复制管理中心在各个可用区对应的代理地址。
+
include::../../_components/note_in_table.adoc[]
当管理中心与所防护的云服务器不在同一个可用区（Zone），则两者之间需要通过代理 IP 进行通信，安装 Agent 时需要把管理中心的地址替换成代理地址。详细操作，请参见link:../../qs/install_agent/[安装 Agent]。

include::../../_components/footer_in_table.adoc[]

|===
