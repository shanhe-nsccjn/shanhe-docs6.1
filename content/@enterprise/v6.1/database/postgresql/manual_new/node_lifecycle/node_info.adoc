PostgreSQL 一个集群可包含多个节点，一个节点即一个数据库计算基本计量单位，对应一台虚拟机和一台存储磁盘。

PostgreSQL 节点目前支持``主实例``、``只读实例``两种类型。

[cols='1,2,2']
|===
| 节点类型 | 说明 | 使用范围

| 主实例
| 可读可写实例。 +
“一主多从”节点。
| 可挂载只读实例。

| 只读实例
| 仅可读实例。 +
最多可创建 5 个可读实例节点。
| 从主实例同步数据，只能与主实例同区域。
|===
