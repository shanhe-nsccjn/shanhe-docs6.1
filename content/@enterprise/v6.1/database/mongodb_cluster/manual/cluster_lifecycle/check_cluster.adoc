MongoDB Cluster 集群创建成功后，可在 AppCenter 查看集群信息，包括集群基本属性、租赁信息、节点信息、配置参数、告警配置、备份信息等。

本小节主要介绍如何查看集群信息。

== 前提条件

已获取管理控制台登录账号和密码，且已获取集群查看权限。

== 操作步骤

. 登录管理控制台。
. 选择**产品与服务** > *数据库与缓存* > *文档数据库 MongoDB Cluster*，进入集群管理页面。
+
可查看当前区域集群列表，以及集群基本信息。
+
image::/images/cloud_service/database/mongodb_cluster/cluster_list.png[集群列表]

. 选择目标集群，点击目标集群 ID，进入集群详情页面。
+
可查看集群各详细信息，以及执行集群的各项功能管理操作。

. 当在对集群执行操作后，可在集群详情页面左下侧，查看集群操作日志。
+
image::/images/cloud_service/database/mongodb_cluster/operate_log.png[操作日志,40%]

=== 基本属性

在集群详情页面左侧**基本属性**区域，可查看集群基本状态、版本信息、节点数量、网络信息等。

点击下拉栏图标，展开集群操作列表，可查看集群服务功能。

image::/images/cloud_service/database/mongodb_cluster/basic_info.png[基本属性,30%]

=== 节点列表

在**节点**页面，可查看集群各节点的 ID、服务状态、监控信息、资源配置等。

image::/images/cloud_service/database/mongodb_cluster/node_list.png[节点]

=== 配置参数

在**配置参数**页面，可查看集群性能优化配置参数项。

include::../../../../../../_components/attention.adoc[]

修改后将导致自动重启集群的参数，请在业务低峰时进行修改。

include::../../../../../../_components/table_footer.adoc[]

image::/images/cloud_service/database/mongodb_cluster/config_list.png[配置参数]

=== 监控告警

在**告警**页面，可以查看集群告警配置信息，及时掌握集群的资源和服务状况。

image::/images/cloud_service/database/mongodb_cluster/alarm_list.png[监控告警]

=== 备份

在**备份**页面，可以查看集群备份列表信息，可查看集群列表和**备份链**结构示意图，以及可从备份创建新集群。

image::/images/cloud_service/database/mongodb_cluster/backup_list.png[备份信息]

=== 租赁信息

在集群详情页面左侧**租赁信息**区域，可以查看集群当前计费信息。
