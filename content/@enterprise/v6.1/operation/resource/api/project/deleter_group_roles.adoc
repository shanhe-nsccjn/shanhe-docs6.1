删除角色组。

== 请求参数

|===
| 参数 | 类型 | 描述 | 是否必选

| action
| String
| DeleteGroupRoles（删除角色组）
| true

| group_roles
| Array
| 角色组 id 列表
| true

| owner
| String
| 所有者
| false

| zone
| String
| 区域 id
| false

| project_id
| String
| 项目 id
| true
|===

== 响应消息

|===
| 参数 | 类型 | 描述

| action
| String
| DeleteGroupRolesResponse

| group_role_ids
| Array
| 角色组 id 列表

| ret_code
| Integer
| 执行成功与否，0 表示成功，其他值则为错误代码
|===

== 示例

=== 请求示例

[,url]
----
https://api.shanhe.com/iaas/?action=DeleteGroupRoles
&group_roles=["gr-gg5l0faz"]
&owner=usr-WantwZJ8
&zone=test
&project_id=pj-xzvlxlb5
----

=== 响应示例

[,json]
----
{
    "action": "DeleteGroupRolesResponse",
    "group_role_ids": [
        "gr-gg5l0faz"
    ],
    "ret_code": 0
}
----
