用户可以通过应用管理查看已安装的应用、分享给自己的应用以及用户自己开发的应用。

== 管理已安装/购买的应用

. 登录管理控制台。

. 选择**产品与服务**> *AppCenter 控制台* > *应用管理*，进入**应用管理** > **安装过的应用**页面。
+
image::/images/cloud_service/appcenter/um_intall_app.png[]

* 您可以在当前页面根据**应用类型**和**状态**查看应用。
+
image::/images/cloud_service/appcenter/mgt_installed_app.png[]

+
* 您还可以在当前页面进行**管理集群**、*管理应用*、*再次购买*、*立即购买*、**提交工单**或者**删除应用**的操作。

. 不同类型应用，可进行的操作略有区别：
+
* 云应用：
+
** 点击**管理集群**可进入集群部署页面（之前未部署过）或集群管理页面（已部署有）。
** 点击**立即部署**进入应用部署页面。

* License 应用：点击**管理应用**进入应用详情，可查看到 License 应用购买记录及交付情况。
* SaaS应用：点击**管理应用**进入应用详情，可对应用实例进行续费、删除、查看资源详情等操作。
+
image::/images/cloud_service/appcenter/saas_app_mgt.png[]

* API应用：

** 点击**管理应用**进入应用详情，可查看到 API 各规格使用量情况、使用状态、API 密钥等信息。
+
image::/images/cloud_service/appcenter/api_use_detail.png[]

** 点击 **API 接口**，进入该 API 应用介绍页，可查看 API 使用文档以及调试 API。
+
image::/images/cloud_service/appcenter/api_userguide_and_debug.png[]

== 管理分享给我的应用

. 登录管理控制台。

. 选择**产品与服务**> *AppCenter 控制台* > *应用管理*，进入**应用管理**页面。
+
image::/images/cloud_service/appcenter/um_app_mgmt.png[]

. 点击**分享给我的应用**，进入**分享给我的应用**页面。
+
image::/images/cloud_service/appcenter/um_share_app.png[]

. 您可以在当前页面根据**应用类型**和**状态**查看别人分享给您的应用。

== 管理我开发的应用

. 登录管理控制台。

. 选择**产品与服务**> *AppCenter 控制台* > *应用管理*，进入**应用管理**页面。
+
image::/images/cloud_service/appcenter/um_app_mgmt.png[]

. 点击**我开发的应用**，进入**我开发的应用**页面。
+
image::/images/cloud_service/appcenter/um_mine_app.png[]

. 您可以在当前页面根据**应用类型**和**状态**查看您开发的应用。

. 您在当前页面可以**测试部署**/*测试购买*、*去 AppCenter 查看*、**查看已创建的集群**或者**提交工单**等。
+

include::../../../../../_components/note.adoc[]
提交工单需要当前云平台支持工单服务功能。
