== 文件存储

[options="header",cols="1,2"]
|===
| 存储类型 | 规格

| 文件存储 EPFS

|===

== 网络

采用高速通信的 IB （InfiniBand，简称 IB）网络，为节点和节点间，节点和存储间带来高达 100Gbps 的高效传输网络，极大的缩短了作业运行周期。
