本章节介绍如何绑定基础 IP。

== 前提条件

* 已获取 {console_name}的账号和密码。
* 云服务器处于"`运行中`"。

== 操作步骤

. 登录 {console_name}。
. 选择**产品与服务** > *计算* > *云服务器*，进入云服务器页面。
. 点击**鼠标右键**，弹出菜单窗口。
+
image::/images/cloud_service/compute/vm/vm_basic_ip.png[]

. 点击**基础网络弹性 IP** > *绑定基础 IP*，完成基础 IP 的绑定操作。
+
在云服务器列表页面，您可以查看云服务器的公网 IP 所在列显示已绑定的基础 IP 地址。
