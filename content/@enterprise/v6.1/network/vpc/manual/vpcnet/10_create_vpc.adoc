本文介绍如何创建一个新的 VPC 网络。

== 操作场景

在 VPC 网络中使用云资源前，您必须先创建一个 VPC 网络和初始私有网络。

== 操作步骤

. 登录管理控制台。
. 在控制台导航栏中，选择**产品与服务** > *网络* > *VPC 网络*，进入**VPC 网络**页面。
. 点击**创建 VPC 网络**，弹出**创建 VPC 网络**页面。

+
image::/images/cloud_service/network/vpc/501010_create_vpc.png[create_vpc]

. 配置 VPC 网络基本信息及初始私有网络信息。
+

include::../../../../../../_components/note.adoc[]

创建 VPC 网络时，应至少创建一个初始私有网络，此外，您还可以单击**添加私有网络**创建多个私有网络，一次最多可添加3个私有网络（包含初始私有网络）。

include::../../../../../../_components/table_footer.adoc[]
+
*基本信息设置：*
+
[cols="1,4"]
|===
|参数名称 |参数说明

|区域 |创建 VPC 网络的区域。
|名称 |VPC 网络的名称。

支持自动填充及自定义。名称长度为1~64个字符，须由中文、英文字母、数字、下划线（_）、中划线（-）和点（.）组成。

|IPv4 地址范围 |VPC 的 IPv4 网段。

VPC 网络创建后，IPv4 网段不可修改。
|IPv4 地址范围 |VPC 的 IPv4 网段。
|IPv6 地址范围 a|选择是否为 VPC 网络分配 IPv6 网段。

* *关闭 IPv6*：默认选项，即不分配 IPv6 网段。
* *系统自动分配*：开启 IPv6 功能，并自动分配 IPv6 网段。
|类型 a|支持免费型、小型、中型、大型及超大型，不同类型可支持的管理流量转发能力不同。 +

include::../../../../../../_components/note_in_table.adoc[]
免费型 VPC 仅具备内网通信功能，不支持绑定公网 IP 和转发公网流量，且暂不支持在免费型 VPC 中创建弹性容器实例(QCI)、AppCenter 集群和边界路由器。 +
若您的云服务器需要上网，需要给私有网络内的云服务器单独绑定公网 IP。

include::../../../../../../_components/footer_in_table.adoc[]

|公网 IP a|选择是否立即为 VPC 绑定公网 IP。免费型 VPC 不支持此项。

* *购买并绑定*：购买一个新的公网 IP 并绑定到此 VPC 网络。选择后需要设置公网 IP 的计费模式、带宽上限及 IP 组。
* *使用已有*：若已购买有公网 IP，可选择绑定已有的公网 IP。
* *暂不绑定*：此处暂不绑定，可在创建 VPC 网络后根据需要进行绑定。具体操作请参见link:/v6.1/network/vpc/manual/vpcnet/26_bind_eip/[绑定公网 IP]。
|安全组 a|选择 VPC 需要绑定的安全组。

若未创建安全组，可选择默认安全组，也可在下方提示信息中点击新建安全组进行创建，具体操作说明参见link:/v6.1/network/security_group/manual/sg_mgt/sg_create/[创建安全组]。
|===
+
*初始私有网络设置：*
+
[cols="1,4"]
|===
|参数名称 |参数说明

|名称 |私有网络名称。
|部署方式 a|* *多可用区部署*：表示网络中的云服务器可部署在多个可用区。

* *单可用区部署*：表示网络中的云服务器部署在一个可用区中，但与其它可用区之间的网络互通。
|IPv4 地址范围 |私有网络的地址范围，需要在 VPC 网络的地址范围内，且私有网络间的 IPv4 网段不可重叠。
|IPv6 地址范围 a|仅当所属 VPC 网络开启 IPv6 功能，其私有网络才支持开启 IPv6。

* *关闭 IPv6*：默认选项，即不分配 IPv6 网段。
* *系统自动分配*：开启 IPv6 功能，并自动为子网分配 IPv6 网段。

如需自定义私有网络的 IPv6 网段，可以在创建完成后先关闭 IPv6，再启用 IPv6 并选择手动指定 IPv6。

|关联路由表 |表示私有网络关联的路由表。默认情况下，VPC 中所有私有网络均关联默认路由表，若需更换路由表，请在创建 VPC 网络后，解绑该路由表并重新绑定。
|网络 ACL |网络访问控制规则。您可以自定义设置网络 ACL 规则，然后绑定私有网络。
|高级选项 |点击**高级选项**，可以设置私有网络的管理地址、边界路由器地址、DHCP 起始地址 及 DHCP 结束地址。一般使用默认设置即可。
|===

. 可选：若需添加多个私有网络，则点击**添加私有网络**，并设置其信息。

+
image::/images/cloud_service/network/vpc/501010_create_vpc_2.png[create_vpc]

. 点击**立即创建**，待创建成功，返回 **VPC 网络**页面。
