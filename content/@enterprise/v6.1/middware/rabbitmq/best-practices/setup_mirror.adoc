支持通过 **Web 界面**和 **HTTP API** 两种方式进行设置。

* **Web 界面**：推荐使用该方式，可视化界面更方便。
* **HTTP API**：若对 RabbitMQ API 比较熟悉，也可以使用 HTTP API 方式。

详细内容请参考link:https://www.rabbitmq.com/ha.html[官方文档]。
