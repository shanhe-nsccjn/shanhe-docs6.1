API 定义录入完成后，您可以在 API 调试页面进行调试，以确定 API 的可用性。

API 调试可通过以下三种途径找到调试按钮进行操作：

* 进入服务的 API 管理页面。
* 在 API 服务菜单中选择 **API** 页签。
* 进入 API 详情页。

== 调试

点击**调试**，页面进入 API 调试页面，页面中会显示 API 的配置信息。

include::../../../../../../_components/note.adoc[]

API 发布后才可以调试。若 API 未发布过，或已下线，调试按钮不可点。且 API 调试时，需依据当前运行 API 配置进行调试。

include::../../../../../../_components/table_footer.adoc[]

. 在发送请求之前，请先填写请求参数。
. 若 API 的鉴权类型为 QingCloud-Auth，调试时会显示密钥栏位，要求用户填写绑定的密钥信息。若绑定多个密钥，填写绑定的一个即可。
. 若 API 创建时在步骤 3 中填写了示例信息，调试页面右侧会显示示例内容。
. 信息配置完成后，点击**发送请求**，请求返回结果会显示在右侧的返回结果栏位。
